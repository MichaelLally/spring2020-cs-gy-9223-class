NYU Motorsports Telemetry


Telemetry platform for monitoring NYU Tandon's Motorsports SAE BAJA competition vehicle. This platform supports reception and visualization of data from the competition vehicle along with several auxiliary features.

Dashboard

Launches a data visualization tool in a new window. This tool queries the team's database to populate several graphs. These graphs are fully configurable and can be added or deleted as new data sources become available.

Stopwatch

Time the race with the Stopwatch feature. Track overall time and individual laps in the browser.

Raw Telemetry

This portal allows the raw data to be viewed. Useful for debugging and understanding the models behind the data visualization portal. Data generated in the simulator and/or vehicle sensors will appear here.

Simulator

Generates mock vehicle data. Allows for experimentation with different data patterns on the visualization portal and raw telemetry portal.

CAN Decoder

Input CAN messages and view the return values. Sends user input to the CAN Decoder API endpoint and returns the decoded JSON values. Useful for learning CAN's format and the differences between structured and CAN data.
